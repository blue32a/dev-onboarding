# Node.js

Node.jsはJavaScriptをブラウザの外で実行するためのランタイムです。サーバーサイドの処理、ファイル操作、CLIツールの作成など、ブラウザでは実現できない幅広い用途に使われています。

このコンテンツでは、Node.jsの基本的な機能をハンズオン形式で体験しながら、サーバーサイドJavaScriptの立ち上がりを支援します。

## 前提

- JavaScriptコンテンツ（01〜10）を完了していること
