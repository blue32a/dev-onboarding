# React

ReactはUIを構築するためのJavaScriptライブラリです。コンポーネントと呼ばれる再利用可能な部品を組み合わせてWebアプリケーションを作ります。

このコンテンツでは、コンポーネントの作成・状態管理・データ取得を通じてReactの基本を学びます。

## 前提

- JavaScriptコンテンツ（01〜10）を完了していること
- HTMLコンテンツ（01〜09）を完了していること
- Node.jsがインストールされていること

## 進め方

このコンテンツには `workspace/` ディレクトリが用意されています。01-setupでReactプロジェクトを作成し、02以降のステップで `workspace/app/` 内のファイルを編集しながら進めます。
