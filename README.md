# 技術オンボーディングコンテンツ

新卒・未経験者など、技術領域の学習を始めたばかりのエンジニアを対象とした、ハンズオン形式の技術学習コンテンツ集です。

## このコンテンツについて

### 目的

各技術領域を「なんとなく知っている」状態ではなく、実務で使える手段として選択できる状態を目指します。具体的には次の3点を目標にしています。

- その技術が何のために必要なのかを理解し、問題解決の手段として選択できる
- 公式ドキュメントなどを参照しながら、自分で学習を進められる
- 不明点を言語化し、適切な質問ができる

### 学び方

各コンテンツはハンズオン形式で構成されています。読むだけでなく、実際にコードを書いて動かし、結果を自分の目で観察することで学びます。

- **前提ファイルは用意済み** — 学習内容に集中できるよう、HTMLなどの土台は提供済みです
- **説明は最小限** — 立ち上がりに必要な概念だけを説明します。詳細は公式ドキュメントへのリンクから参照できます
- **コードはファイルに書く** — Console での使い捨てではなく、ファイルにコードを書くことで学習記録として残ります

### 使い方

1. 各技術のディレクトリにある `README.md` からステップ順に進めます
2. ステップごとにコードを書きながら動作を確認します
3. わからないことは各ステップの「一次情報・参考資料」から公式ドキュメントを参照します

| [React](./react/README.md) | UIを構築するためのJavaScriptライブラリ。コンポーネントベースでWebアプリケーションを開発する |
