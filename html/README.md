# HTML

HTMLはWebページの構造を定義するマークアップ言語です。テキスト・リンク・フォーム・画像などの要素を組み合わせ、ブラウザが表示できるドキュメントを作ります。

このコンテンツでは、HTMLの基本構造からセマンティクス・フォーム・アクセシビリティまで、ブラウザで確認しながら学びます。

## 前提

- JavaScriptコンテンツ（01〜10）を完了していること
- VS Codeがインストールされていること
