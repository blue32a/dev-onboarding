# JavaScript

JavaScriptはWebブラウザ上で動作するプログラミング言語です。HTMLで構造を作り、CSSでスタイルを整えたページに対して、JavaScriptは動きと機能を加えます。ボタンへの反応、外部APIからのデータ取得、ブラウザへのデータ保存など、フロントエンド開発の中心的な役割を担います。

このコンテンツを通じて、ブラウザを実行環境として使いながら、実務で頻繁に使うJavaScriptの要素を順に体験していきます。
